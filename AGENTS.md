## Cursor Cloud specific instructions

This repository is an **Obsidian vault** (personal knowledge management / notes repository), not a software project.

### Key facts
- **No source code** exists outside `.obsidian/` plugin bundles.
- **No package manager**, build system, test framework, or runnable application is present.
- **No dependencies to install** — there is no `package.json`, `requirements.txt`, `pyproject.toml`, or equivalent.
- **No dev server, tests, or linting** to run.
- The `.obsidian/` directory contains Obsidian app configuration, including the `obsidian-git` plugin used for Git-based syncing.

### Content structure
| Directory | Contents |
|-----------|----------|
| `Business/` | Business planning, product strategy, fundraising, sales, and design notes for "DiscoPay" |
| `Personal/` | Personal notes (finances, health, education, interests) |
| `Diary/` | Daily journal entries |
| `Archive/` | Archived notes and images |
| Root files | CSV/XLSX transaction simulation data, meeting transcripts (TXT), PDFs |

### For future agents
- There is nothing to build, lint, test, or run in this repository.
- The only tool associated with this repo is Obsidian (a desktop Markdown editor).
- Any code-related work would need to be scaffolded from scratch.
