## Bubble glossary highlights

- Option sets: predefined choices without storing each option as a database row.
- Conditional expressions: yes/no logic controlling workflows and UI changes.
- Workflows: trigger + action chains (e.g., click event -> navigate/update data).
- Data types: structured containers made of fields.
- Input forms: elements that collect user input.
- Custom states: temporary UI/app data not persisted to the database.
