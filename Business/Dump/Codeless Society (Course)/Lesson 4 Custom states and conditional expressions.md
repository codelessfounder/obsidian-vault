**Objective**: Gain an understanding of how to use custom states and conditional logic to manage dynamic content and app behavior.

- **Duration**: 1 hour
- **Content**:
    1. **What are Custom States?**:
        - Explain custom states and how they differ from the database.
        - Use cases: temporary data, UI element states, etc.
    2. **Conditional Logic**:
        - Discuss conditional logic for elements and workflows (e.g., showing/hiding elements based on conditions).
    3. **Hands-on**:
        - Build an interactive counter: Create a button that increases a custom state value on click and displays the updated value in real-time.
        - Add conditions (e.g., display a message when the counter reaches a specific value).

**Assessment**: Employees should be able to use custom states for real-time interactivity and apply basic conditional logic.
