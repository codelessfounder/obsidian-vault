**Objective**: Understand Bubble's responsive engine and create a mobile-friendly app layout. Implement finishing touches such as styling and testing the app.

- **Duration**: 1 hour
- **Content**:
    1. **Responsive Design Concepts**:
        - Explain responsive design and how Bubble handles responsiveness.
    2. **Using Bubble’s Responsive Engine**:
        - How to design layouts that adapt to different screen sizes (using columns, row layouts, and responsive settings).
    3. **Basic Styling**:
        - Using Bubble’s styles, fonts, and colors.
    4. **Hands-on**:
        - Make the contact list app responsive.
        - Add finishing touches: consistent styling, margins, and paddings to make the app visually polished.

**Assessment**: Employees should have a fully responsive app by the end of this lesson.
