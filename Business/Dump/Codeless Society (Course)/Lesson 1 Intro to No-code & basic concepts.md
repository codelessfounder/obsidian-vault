### **Lesson 1: Introduction to No-Code Development & Bubble Basics**

**Objective**: Understand the concept of no-code development and get familiar with Bubble's interface and structure.

- **Duration**: 1 hour
- **Content**:
    1. **Introduction to No-Code**:
        - Explain what no-code development is and how it fits into modern app development.
        - Discuss Bubble’s role in no-code and why it’s a popular choice for building web apps.
    2. **Bubble Dashboard Tour**:
        - Walk through the Bubble editor: Design tab, Workflow tab, Data tab, Plugins, Settings.
        - Explain the concept of pages, elements, and the responsive engine.
    3. **Building Blocks Overview**:
        - **Elements**: Basic visual elements (text, button, input fields).
        - **Pages**: How pages work in Bubble, building a multi-page app vs. single-page.
        - **Workflows**: Introduction to workflows and event-driven logic.
    4. **Hands-on**:
        - Build a simple landing page with text and a button. Add a basic action (like an alert) using workflows.

**Assessment**: Employees should be able to create a basic page layout and use a simple workflow by the end of this lesson.
