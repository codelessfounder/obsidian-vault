**Objective**: Understand how Bubble manages data and how to create and manipulate databases.

- **Duration**: 1 hour
- **Content**:
    1. **Introduction to Bubble’s Database**:
        - Explain Data Types (Tables) and Fields (Columns).
        - Creating a new data type (e.g., “User” with fields such as Name, Email, and Age).
    2. **Dynamic Data vs. Static Data**:
        - Explain the difference between static data (hard-coded) and dynamic data (from the database).
    3. **Displaying Data on a Page**:
        - Using repeating groups to display lists of data.
    4. **Hands-on**:
        - Create a simple contact list app: Set up a "Contact" data type and build a repeating group to display a list of contacts.
        - Add inputs for creating new contacts and a submit button to add entries to the database.

**Assessment**: Employees should successfully create a working database, add items to it, and display the data using repeating groups.
