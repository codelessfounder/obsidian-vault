**Objective**: Learn how to build a user registration, login, and authentication system using Bubble.

- **Duration**: 1 hour
- **Content**:
    1. **User Authentication Overview**:
        - Explain how Bubble’s built-in authentication works.
    2. **Setting Up User Accounts**:
        - Creating sign-up, login, and logout workflows.
    3. **Access Control**:
        - How to manage data access based on user roles and login status.
    4. **Hands-on**:
        - Build a user sign-up and login form.
        - Use conditions to display personalized content (e.g., show the user’s name on the homepage after login).

**Assessment**: Employees should be able to create a functional user registration and login system and understand basic access control.
